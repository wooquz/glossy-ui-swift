# GlossyUI

Modern SwiftUI library bringing elegant glass morphism effects to iOS applications.

## Features

- Beautiful translucent card components
- Customizable blur and glow effects  
- Smooth gradient overlays
- Easy SPM integration
- iOS 15+ support

## Installation

### Swift Package Manager

## Usage

Import the library and wrap your content with GlossyCard view. Customize appearance using init parameters like cornerRadius, blurIntensity, and glowOpacity.

## Requirements

- iOS 15.0+
- Swift 5.7+
- Xcode 14.0+
